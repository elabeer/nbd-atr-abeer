# نبض الروح

موقع Landing Page ثابت للتعريف بخدمات العلاج النفسي والاستشارات النفسية أونلاين.

## نظرة عامة

- HTML5 فقط
- CSS3 فقط
- Vanilla JavaScript فقط
- Font Awesome 6 (fas)
- Google Fonts Cairo
- بدون أي إطار عمل أو مكتبات خارجية
- يعمل بمجرد فتح `index.html`
- جاهز للنشر على GitHub Pages

## كيفية النشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub.
2. ارفع جميع الملفات مع الهيكلية الحالية.
3. قم بتفعيل GitHub Pages من إعدادات المستودع واختَر الفرع `main` أو `gh-pages`.
4. سيعمل الموقع مباشرة بدون أي إعدادات إضافية.
